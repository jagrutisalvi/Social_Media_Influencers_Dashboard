# 📊 Social Media Influencers Dashboard (Tableau)

## 📌 Overview
This project analyzes social media influencers’ data to understand engagement, follower growth, and overall performance. An interactive dashboard is created using Tableau to visualize key insights.

## 📁 Data Source
- Dataset in CSV format
- Data includes influencer metrics such as followers, engagement, likes, and reach

## 🛠 Tools & Technologies
- Tableau (data visualization and dashboard)
- CSV Dataset

## 🔍 Key Analysis
- Engagement rate analysis
- Follower growth trends
- Influencer performance comparison
- Content reach and interaction insights

## ⚙️ Process
1. Collected and loaded dataset into Tableau  
2. Cleaned and prepared data  
3. Created calculated fields (if any)  
4. Built interactive dashboard with filters and visuals  

## 📂 Project Files
- Social_Media_Influencer_Dashboard / .twbx → Tableau dashboard file  
- social_media_influence_project_dataset.csv → Dataset used for analysis  

## 📊 Insights
- Identified top-performing influencers based on followers  
- Observed trends in follower growth  
- Compared performance across different influencers  

## 📌 Conclusion
This dashboard helps in understanding influencer performance and can support better marketing and collaboration decisions.
